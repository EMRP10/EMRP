Release Note

We (CROP authors) are currently evaluating the best way of releasing the CROP dataset given the new EU GDPR regulation. Stay tuned to this website for any development on the CROP release.

In case of interest in using the CROP data at this moment, we strongly encourage the CROP users to personally contact the main CROP maintainer Matheus Paixao at matheus.paixao.14 (at) ucl.ac.uk
